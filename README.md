# 🥗 Nutriela

**Nutriela** é uma Progressive Web App (PWA) desenvolvida para auxiliar usuários no acompanhamento nutricional diário, oferecendo cálculos metabólicos automáticos, monitoramento de macronutrientes e funcionamento totalmente offline.

O projeto foi desenvolvido com foco em acessibilidade, praticidade e experiência mobile, permitindo que o usuário acompanhe sua alimentação sem depender de conexão com a internet.

---

## 🚀 Principais Funcionalidades

### Perfil Nutricional

* Cadastro de nome, idade, peso e altura
* Seleção de gênero
* Definição de objetivo físico:

  * Perda de peso
  * Manutenção de peso
  * Ganho de massa muscular
* Configuração do nível de atividade física

### Cálculos Automáticos

* Taxa Metabólica Basal (TMB)
* Gasto Energético Diário Total (TDEE)
* Índice de Massa Corporal (IMC)
* Metas calóricas personalizadas
* Distribuição automática de macronutrientes
* Meta diária de hidratação

### Registro Alimentar

* Cadastro de refeições
* Banco de alimentos integrado
* Controle de proteínas, carboidratos e gorduras
* Histórico diário de consumo

### Dashboard Nutricional

* Progresso calórico em tempo real
* Acompanhamento de macronutrientes
* Controle de ingestão de água
* Visualização simplificada das metas diárias

### Funcionamento Offline

* Armazenamento local utilizando LocalStorage
* Cache inteligente através de Service Worker
* Instalação como aplicativo no celular
* Utilização sem conexão com internet

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido para consolidar conhecimentos em desenvolvimento web, Progressive Web Apps e manipulação de dados no navegador.

Além do aspecto técnico, o Nutriela busca oferecer uma solução simples e acessível para acompanhamento nutricional diário.

---

## 📱 Instalação

1. Acesse a aplicação pelo navegador.
2. Clique em "Instalar Aplicativo" ou "Adicionar à Tela Inicial".
3. Utilize normalmente, mesmo sem conexão com a internet.

---

## 🔒 Privacidade

Todos os dados permanecem armazenados localmente no dispositivo do usuário.

Nenhuma informação é enviada para servidores externos.
